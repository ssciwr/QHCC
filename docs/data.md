## Pooch

[Pooch](https://www.fatiando.org/pooch/latest/) is a Python library for downloading datasets,
and it has native support for heiData. To install:

=== "pip"
    ```bash
    pip install pooch
    ```

=== "conda"
    ```bash
    conda install pooch
    ```
